# Ratakierros iOS

Native iOS app (Swift / SwiftUI). MVP scope:

- Show the closest athletics track using the device's location.
- Register / log in with email + password.
- Log a run time on the closest track.
- View the per-track leaderboard (top 10) and personal best.

Talks to the API at `https://ratakierros.fi` (overridable via the
`API_BASE` key in `Info.plist`).

## Build locally

Requires:

- Xcode 15+ (iOS 17 SDK)
- [XcodeGen](https://github.com/yonaskolb/XcodeGen):
  `brew install xcodegen`

Then:

```sh
cd ios
xcodegen generate
open Ratakierros.xcodeproj
```

Pick an iPhone simulator and run. On first launch the app asks for
location permission — grant it to see the closest track.

To build from the command line (e.g. on CI):

```sh
xcodegen generate
xcodebuild \
  -project Ratakierros.xcodeproj \
  -scheme Ratakierros \
  -destination "platform=iOS Simulator,name=iPhone 15,OS=latest" \
  -skipPackagePluginValidation \
  CODE_SIGNING_ALLOWED=NO \
  build
```

## Layout

```
ios/
├── project.yml                       # XcodeGen manifest
├── Generated/Info.plist              # generated by xcodegen
├── Ratakierros.xcodeproj/            # generated by xcodegen (gitignored)
└── Sources/Ratakierros/
    ├── RatakierrosApp.swift
    ├── Models/{Track,Run,AuthResponse}.swift
    ├── Networking/APIClient.swift
    ├── Auth/{AuthStore,LoginView}.swift
    ├── Location/LocationProvider.swift
    └── Views/{ClosestTrackView,LogRunView,LeaderboardView}.swift
```

`AuthStore` persists the JWT in the iOS Keychain (`fi.ratakierros.app`
service, `jwt` account).
